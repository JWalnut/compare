Corpus¶

A big hefty wrapper class

class compare.classes.corpus.Corpus(self, folder = "", title = "")¶

A class to wrap many of the document comparison methods, so that comparisons may be done multiple times on the same set of data.

Fields¶

compare.classes.corpus.folder¶

Holds the path to the folder containing all of the comparable files

compare.classes.corpus.title¶

A simple identifier for the object. Defaults to equal folder

compare.classes.corpus.doclist¶

A list of Document objects containing the documents to be compared

compare.classes.corpus.wordlist¶

A list of all of the words in all of the given documents

compare.classes.corpus.corpuswordmap¶

A NetworkX graph to hold the wordmap of the folder of documents

compare.classes.corpus.sparsemap¶

Subgraph of corpuswordmap containing only those nodes that connect to other nodes. That is, excluding nodes that only map to themselves.

compare.classes.corpus.markov¶

A NumPy array to hold the Markov matrix generated from the folder of documents

compare.classes.corpus.meta¶

A structure holding metadata about the documents to be compared

compare.classes.corpus.addFolder(self, foldertoadd = "", quiet = True, drawMap = False, synCount = False)¶

Loads a folder of documents to be compared into the object, and generates a wordmap from those documents and the markov matrix.

Parameters:
  • foldertoadd (String) – Path to the folder containing comparable documents
  • quiet (Boolean) – Toggle status bar when generating wordmap
  • drawMap (Boolean) – Toggle graphical drawing of generated wordmap
  • synCount (Boolean) – Toggle graph weights based on how frequently WordNet gives it as a synonym
compare.classes.corpus.compareCorpus(self, depth = 0, count = False)¶

Compares all of the documents loaded into the object, and returns a matrix of their pairwise distances See wordmap.compare()

showMap(self, allNodes=True):

Uses matplotlib to display the wordmap.

Parameters:allNodes (Boolean) – If “allNodes” is True, displays corpuswordmap. If false, sparsemap
