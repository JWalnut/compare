static compare.func.wordmap.generateDocumentVector(allWords, document, count=False)¶

Generates a vertical vector of the frequency of occurance in allWords of each word in document The order of the words appearing in the returned vector will match the order of the words appearing in allWords.wordlist

If parameter “count” is true, the vector contains the actual frequence (i.e., if “test” appears three times, its entry will be “3”) If parameter “count” is false, the vector ignores duplicate occurances (i.e., “test“‘s entry would be “1”)

Parameters:
  • allWords (Document) – Document contating all of the words in the two documents being compared
  • document (Document) – Document from which to generate the vector
  • count (Boolean) – Flag to determine whether the vector should contain the actual count of each word
Returns:

Vertical vector
Return type:

NumPy Matrix
